Display a Strobe org's current events directly through WordPress.

== Description ==

Anyone with a Strobe account can use this plugin! The plugin will allow you to display all current events listed on Strobe directly t a WordPress-powered site or blog.

*   no development required!
*   sales information stays up to date and consistent
*   matches your site's style

== Installation  ==

1. Select `Download` from the WordPress plugins directory.
2. Visit the Strobe Dashboard [org settings](https://dashboard.strobelabs.com/org) page to find a secure access token.
3. Copy the access token.
4. Navigate to the admin settings panel for the **Ticketing Widget** plugin.
5. Paste the access token in the designated input box.
6. Add a `[strobe_event_list]` short code wherever you'd like the list to show up on your site.

== Frequently Asked Questions ==

= What do I need to do to use this plugin? =

Copy your access token from the [Strobe Dashboard](https://dashboard.strobelabs.com/org) and paste it in the plugin Settings on your WordPress site.

= Where do I find an access token for my org? =

Visit your [org settings](https://dashboard.strobelabs.com/org) page to find an access token.
